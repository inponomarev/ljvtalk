PriorityQueue: what’s the trick?

  • Binary min-heap, invariants:

    • \(q[n] \leq q[2n + 1]\)

    • \(q[n] \leq q[2n+2]\)

  • Consequence:

    • \(q[0] = \min(q)\)

TreeMap: red-black tree

Red-black tree: What’s the trick?

  • Red-black tree invariants:

    • The root is black.

    • the "`null` leaves" at the ends of branches are considered black.

    • Both children of every red node are black.

    • Every simple path from a parent node to a descendant leaf node has the same number of black nodes.

  • Consequence:

    • The path from the root to the farthest leaf (BRBRBR…​) is at most twice as long as to the nearest leaf (BBB…​).
